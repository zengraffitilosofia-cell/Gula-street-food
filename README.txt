Street Food Gula - versión estática editable

Archivos principales:
- index.html -> home
- menu.html -> carta interactiva
- styles.css -> estilos globales
- script.js -> acordeón del menú
- assets/ -> logo e imágenes

Puedes abrir index.html directamente en el navegador.
Para editar:
- textos y estructura: index.html y menu.html
- colores, tamaños, espaciados: styles.css
- imágenes: carpeta assets
